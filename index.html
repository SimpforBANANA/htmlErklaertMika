HTML - alle Tags an einem Ort

Hier findest du alles, was du über HTML wissen musst

Generelle Informationen

HTML steht für Hyper-text-markup-language. Es ist die Sprache mit der man nicht nur den Text einer Website schreibt, sondern auch bestimmte Befehle in Form von sogenannten Tags. Dateien in diesem Format endet in .html oder in alten betriebssystemen .htm
Man kann aber auch einfach nur Text in die Datei schreiben, welcher dann auch als solches im Browser angezeigt wird, allerdings bleiben einem dann alle Möglichkeiten den Text zu Formatieren und der Quellcode sieht nicht sehr geordnet aus.


Zu den Tags

Jetzt Zeige ich euch alle wichtigen Tags mit Beispielen

Der h tag

Der h (Header) tag wird benutzt um Überschriften in verschiedenen Größen zu schreiben. h1 ist die größte Überschrift und h6 die kleinste. So benutzt du ihn: 


<h1>Dies ist eine sehr große überschrift</h1>    
<h3>Diese überschrift ist etwas kleiner</h3>
<h6>Und diese hier ist die kleinste Überschrift</h6>

Der p tag

Der p (paragraph) tag wird benutzt um lange Stücke normalen Text zu schreiben. Dieser Satz ist mit dem p tag geschrieben. Und So benutzt du ihn:


<p>So einfach geht Text!</p>

Die formatierungs-tags

Um text zu formatieren (fett, schräg, unterstrichen) kann man verschieden tags benutzen. Man kann damit auch einzelne Worte aus einem Satz herausheben


<p>Dieser Text ist Normal, aber diese Worte sind <b>fett</b> oder <i>schräg</i> oder auch <u>unterstrichen</u>

Aufzählungen

Erstmal gibt es zwei Arten von Aufzählungen:

  1. Geordnete Aufzählungen (so wie hier)
  2. Ungeordnete Aufzählungen (mit spiegelstrichen)

Am Anfang muss man ein einleitenden tag benutzen, um die Art der Liste zu bestimmen: für eine Geordnete Aufzählung ol (ordered list) und für eine ungeordnete Aufzählung ol (unordered list). Danach benutzt man für jeden Punkt den li tag.

So geht's:

Geordnete Aufzählungen


<p>Beste Früchte</p>

<ol>
    <li>Mango</li>
    <li>Wassermelone</li>
    <li>Mandarine</li>
</ol>

Ungeordnete Aufzählungen


<p>Liste von Früchten</p>

<ul>
    <li>Apfel</li>
    <li>Orange</li>
    <li>Kirsche</li>
<ul>

Hyper-referenzen

Um von deiner Seite auf eine andere zu verlinken, benötigst du Hyper-referenzen. Diese benutzen den a tag, welcher mindestens das href argument benötigt, in dem man den link zur Website einfügt. Dies funktioniert mit Seiten im www aber auch relative Bezüge (link zu einer html Datei, die im gleichen Ordner gespeichert ist, wie die Hauptseite).

So funktionierst:


<p>Unsere Schulwebsite ist <a href="zip.itslearning.com">itslearning</a></p>
<p>Mehr infos dazu: <a href="./Infos">Hier</a></p>
